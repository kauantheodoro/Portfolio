Detalhes do Projeto 2

Projeto Nome: Nitt

Descrição: O projeto consiste na criação de um website para gerenciamento de um sistema de pedidos, clientes e produtos, utilizando um banco de dados relacional. As principais etapas incluem a implementação do banco de dados físico, a criação de funcionalidades para inclusão, alteração e exclusão de dados, a elaboração de consultas específicas, e a geração de relatórios em formato PDF. O website será composto por uma interface intuitiva que permitirá aos usuários realizar operações como cadastrar novos clientes, produtos e pedidos, consultar informações específicas, e gerar relatórios detalhados com base nos dados armazenados no sistema.

Data de Conclusão: 10 de Novembro de 2024

Tecnologias Utilizadas: Php - Html
